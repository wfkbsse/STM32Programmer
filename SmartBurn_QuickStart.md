# 智能自动烧录 - 快速开始

## 🚀 3步开始使用

### 步骤1: 准备固件
```
1. 点击"自动搜索BOOT"按钮
2. 点击"自动搜索APP"按钮
3. 确认固件已选择 ✅
```

### 步骤2: 启动智能烧录
```
1. 点击"智能自动烧录"按钮（绿色）
2. 在确认对话框中点击"是"
3. 看到"等待下一块板子..."提示 ✅
```

### 步骤3: 开始烧录
```
1. 连接第一块板子
2. 等待自动烧录（无需点击）
3. 听到提示音 → 移除板子
4. 重复步骤1-3
5. 完成后点击"停止自动烧录"
```

## 🔊 声音提示

- **叮～** = 烧录完成，请移除板子
- **咚！** = 烧录失败，请检查

---

## 💡 提示

- ✅ 全程无需点击，只需放置和移除板子
- ✅ 实时显示已完成数量
- ✅ 失败会自动提示，不影响后续烧录
- ✅ 随时可以点击"停止"按钮结束

```
连接板子 → 点击"一键烧录" → 等待 → 点击确认 → 移除板子
需要: 2-3次点击

```
